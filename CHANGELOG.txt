CHANGELOG
=========

2026-03-28  ABEL-HDL frontend (third language)
  - Lexer and parser for ABEL-HDL (Data I/O / Synario, 1995)
  - Definition file: 37 keywords, 36 operators (abel_tok.def)
  - Case-insensitive keyword matching per ABEL spec
  - Comments: " (to matching " or EOL) and //
  - Numbers: ^b binary, ^o octal, ^d decimal, ^h hex
  - Special constants: .X. .C. .D. .F. .K. .P. .U. .Z.
  - Sections: MODULE, DECLARATIONS, EQUATIONS, TRUTH_TABLE,
    STATE_DIAGRAM, TEST_VECTORS, END
  - Pin/node declarations with ISTYPE attributes
  - Dot extensions for signal attributes (.CLK, .OE, etc)
  - WHEN-THEN-ELSE and IF-THEN-ELSE in equations
  - State diagram parsing with transitions and output equations
  - Auto-detection from .abl file extension
  - Reference manual retrieved from bitsavers.org before the
    last copy disappeared. Every university link is dead.
  - 6 new tests (76 total), example decoder.abl design
  - Three frontends: SystemVerilog, VHDL, ABEL-HDL

2026-03-28  CI
  - GitHub Actions on Windows, Linux, macOS
  - Fixed POSIX popen for Linux, clang warnings for macOS
  - Test buffer bumped to 64KB for cross-platform consistency

2026-03-25  Radiation hardening (TMR)
  - --tmr triplicates all DFFs and inserts majority voters
  - --tmr-full triplicates all cells including combinational
  - Voter is standard AND/OR (no new cell types)
  - Works with all backends: ASIC, FPGA, all PDKs
  - Voyager FDS: 15 DFFs -> 45 DFFs + 15 voters (172 cells)
  - Minuteman D17B: 11 DFFs -> 33 DFFs + 11 voters (120 cells)
  - Pool overflow detection with graceful failure

2026-03-25  Package support, typedef enum width propagation
  - package ... endpackage parsed at top level
  - import pkg::* and import pkg::item accepted
  - typedef enum with explicit base type (logic [N:M]) now
    propagates bit width to all ports and signals using that type
  - enum values (IDLE=0, RUN=1, etc.) extracted and stored
  - tname registry extended: width, enum names, enum values
  - width resolver checks user-defined type names before defaulting
  - 70 tests pass, all 10 designs at zero errors

2026-03-24  FPGA targeting, equivalence checker, SV features
  - --fpga <f> emits nextpnr JSON for iCE40 (4-LUT mapping)
    Setun ALU: 9 LUTs. Voyager FDS: 16 LUTs, 9 DFFs.
  - --equiv combinational equivalence checking
    Exhaustive (formal proof) for <=24 input bits, random for larger
    Voyager FDS and Setun ALU formally proved equivalent
  - typedef struct packed { ... } name_t parsed (brace tracking)
  - C-style i++/i-- in for loops desugared to i = i + 1
  - function/endfunction skipped gracefully
