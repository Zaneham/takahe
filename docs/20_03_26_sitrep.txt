COMPARISON
----------
vs Synopsys DC: Missing boolean minimisation, timing-driven opt,
physical awareness. Has better timing arithmetic (exact int64
vs IEEE 754 double). Has multi-radix (13 paradigms vs 1).
Price: free vs $200k/yr.

vs Yosys: Yosys is an elaborator that shells out to ABC.
Takahe is a standalone synthesis tool with built-in STA.
No dependencies beyond libc.

vs Nothing: There is no open-source tool that does synthesis
with timing analysis in one binary. Takahe is the first.
